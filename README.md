# 実質賃金計算機

名目賃金と消費者物価指数（CPI）から、物価変動を除いた実質的な購買力＝**実質賃金**を計算するWebツールです。AI開発支援ツール（Cursor / Claude Code）を活用し、早稲田大学3年（文化構想学部）在学中に個人で企画・実装・公開しました。

🔗 **公開URL: https://selenium132.github.io/jisshitsu-chingin**

## 何ができるか
- 名目賃金と任意の2時点を選び、実質賃金（物価変動を除いた購買力）を計算
- 総務省「消費者物価指数」（2020年基準・1970〜2025年）を実データで搭載
- 賃金が実質で増えたか・減ったかを可視化

## 技術・工夫
- **構成**: HTML / CSS / JavaScript の単一ファイル（外部ライブラリ依存なし・高速・サーバー不要・完全クライアントサイド）
- **実装**: AI開発支援ツールで実装し、CPIデータを総務省の公表値と照合して検証（AIの出力をうのみにしない）
- **公開・運用**: Netlify でホスティング、Google Search Console 登録、sitemap.xml / robots.txt / canonical を整備
- **UX**: スマートフォン最適化（レスポンシブ）

## 意識したこと
- 経済ニュースで話題の「実質賃金」を、専門知識なしで自分の数字を試せるUIにすること
- AIを道具に高速で作りつつ、出力を一次情報で検証して正確性を担保すること
- 作って終わりにせず、公開・SEO・運用まで一通りやり切ること
